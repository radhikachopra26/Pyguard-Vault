# 🛡️ PyGuard Vault 🐍

[![Python Version](https://img.shields.io/badge/python-3.8+-green.svg)](https://www.python.org/downloads/)
[![Security](https://img.shields.io/badge/Security-Validated-brightgreen.svg)]()

A lightweight security tool to generate high-entropy passwords and validate their strength using pattern matching.

---

## ✨ Features

* 🔑 **Secure Generation:** Creates random passwords with letters, numbers, and symbols.
* 🔍 **Strength Analysis:** Uses Regex to check for complexity (Length, Uppercase, Numbers, Special characters).
* ⚡ **Interactive CLI:** Simple menu-driven interface for easy use.
* 🛡️ **Privacy Focused:** Does not save passwords to any file or cloud (Local use only).
